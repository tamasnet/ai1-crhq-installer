# ai1-crhq-installer

A CRHQ skill that installs bundles of CRHQ resources (skills, agents, recipes) and standalone services (nginx + PM2 web apps) into a CRHQ satellite.

See [`SKILL.md`](./SKILL.md) for full documentation.

## Status

Under development. Not yet installed into the local satellite.
